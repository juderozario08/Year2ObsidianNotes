

**Bivariate Population**:  Each experimental unit is characterized with two variables.

# Population Size and Sample Size
The number of experimental units in the sample is equal to the number of values of one variable in the sample. 

The values of **continuous variables** result from the process of measuring. 
Since the values of continues var are real numbers, continuous variables assume infinite number of values even within finite interval on the computer. 

When the value of the variable theoretically may be obtained with any degree of precision, and is represented by real number, the variable is considered as continuous variable. 

# Parameters and Statistics
Population and chars can be characterized with certain numbers. 

***Parameters***: Numerical characteristic of a population. Eg. *Population mean, Population variance and population proportion*

***Statistic***: Numerical characteristic of a sample. *Sample proportion expressed as a percent*.

# Graphs of Categorical Data and Numerical Data
*Summary tables, bar charts and pie charts are for categorical data sets.* 
*Frequency distributions, relative frequency distributions and histograms for numerical data sets.* 

**Categorical (Qualitative) data set**: Set of vales of categorical variable. Total number of categories is denoted by k.

**Dotplots (Quantitative)**:  Used for small data sets.
**Stem and Leaf (Quantitative)**: 
	- Divide each number in the data set into two parts: stem and leaf.
	- Arrange the stems in ascending order and place the stems in a column.
	- For each number, place the leaf portion in the same row as its corresponding stem. 
	- Order the leaves from lowest to highest in each stem.
	- Indicate the leaf unit.
