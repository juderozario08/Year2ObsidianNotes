# For Loops
```java
    // start, stop, step
for (int i = 0; i < 10; i+=1){
	do something;
}
```
# While Loops
```java
int i = 0;
while (i < 10){
	do something;
	i+=1;
}
```

# Do While Loops
- This makes sure that there is always at least one iteration of a task that's being done whether our condition matches or not
```java
do {
	do something;
} while (i < 10)
```
